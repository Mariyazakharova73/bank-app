### app-bank

#### Технологии

React, react-router-dom, scss, БЭМ

#### О проекте

Приложение банка

#### Установка и запуск приложения

Клонировать репозиторий (frontend):

    git clone https://github.com/Mariyazakharova73/bank-app.git

Установить зависимости:

    npm i

Запустить проект:

    npm run dev

Запустить eslint:

    npm run lint

Запустить тесты:

    npx vitest
